## 6. Configuration Reference

### 6.1 next.config.js

| Property | Value | Description |
|----------|-------|-------------|
| `output` | `'standalone'` | Produces minimal production build |
| `images.remotePatterns` | Array | Allowed external image domains |

### 6.2 tailwind.config.ts

| Property | Description |
|----------|-------------|
| `content` | Paths to scan for class usage |
| `theme.extend.colors` | Custom color palette |
| `theme.extend.fontFamily` | Custom fonts (Inter, Plus Jakarta Sans) |
| `theme.extend.animation` | Custom animations |

### 6.3 docker-compose.yml

| Property | Value | Description |
|----------|-------|-------------|
| `ports` | `3001:3000` | External:Internal mapping |
| `restart` | `unless-stopped` | Auto-restart policy |
| `healthcheck.interval` | `30s` | Health check frequency |
| `networks` | `nervescape-network` | Isolated bridge network |

## 7. Security Considerations

| Layer | Security Measure |
|-------|------------------|
| **Container** | Non-root user (nextjs) |
| **Network** | Isolated Docker bridge network |
| **Images** | Whitelisted external domains only |
| **Build** | Multi-stage build (no dev deps in prod) |
| **Health** | Automated health checks |

---

## 8. Performance Optimizations

| Optimization | Implementation |
|--------------|----------------|
| **Code Splitting** | Automatic per-route bundles |
| **Image Optimization** | Next.js Image component |
| **CSS Purging** | Tailwind removes unused styles |
| **Standalone Build** | Minimal production output |
| **Alpine Image** | Smallest base image (~5MB) |

```bash
npm run dev      # Start development server
```

```bash

docker compose down     # Stop container
```
